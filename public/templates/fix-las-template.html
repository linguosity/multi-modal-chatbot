Fixing the LAS Assessment Report Template

Error Found: Your template contains text with curly braces {...} that docxtemplater is treating as an unclosed tag.

The error message was:

Unclosed tag: The tag beginning with "{II. BACKG" is unclosed

Step 1: Open the Template in Word

Open the las-assessment-report-template.docx file in Microsoft Word or another word processor.

Step 2: Locate the Problem

Look for text like {II. BACKGROUND INFORMATION} - these are not intended to be template tags but normal text that happens to be in curly braces.

Based on the error, the issue is with text containing {II. BACKGROUND...

Step 3: Fix the Issue

Choose one of these methods:

Method A: Escape the Braces

Add a backslash before each curly brace that isn't meant to be a template tag:

Original Fixed
{II. BACKGROUND INFORMATION} \{II. BACKGROUND INFORMATION\}
{III. ASSESSMENT RESULTS} \{III. ASSESSMENT RESULTS\}

Method B: Use Custom Delimiters

At the beginning of your document, add this tag to change the delimiters:

{=<% %>=}

Then write all your actual template tags using these new delimiters:

Original Template Tag New Format
{studentFullName} <%studentFullName%>
{#strengthsList}{text}{/strengthsList} <%#strengthsList%><%text%><%/strengthsList%>

At the end of your document, you can switch back:

<%={ }=%>

Step 4: Save and Test

Save your changes and test the export again.

TIP: Method B (custom delimiters) is generally easier if your document has many section headers with roman numerals in curly braces like {I.}, {II.}, etc.

Alternative: Create a New Template

If fixing the template is too difficult, you can also:

  1. Create a new document without problematic curly braces
  2. Add template tags according to the README.md instructions
  3. Save it as las-assessment-report-template-fixed.docx
